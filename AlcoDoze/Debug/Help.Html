О программе

Программа предназначена для расчета концентрации алкоголя в крови. Информация о расчетах, используемых данных отражена в файле информации.

Работа программы

При входе в программу, открывается меню. При нажатии на кнопку Начать тест, осуществляется переход на форму с расчетами. При нажатии на кнопки Информация, Справка, открвываются соответствующие html файлы. Для начала самих расчетов необходимо заполнить поля, при наведении на поля отображается информация о вводе.
