Выравнивание по левому краю

Строки прижимаются к левому краю окна и образуют неровный правый край.

Выравнивание по правому краю

Строки текста выровнены по правому краю, а левый край получается "рваный". Поскольку мы привыкли читать текст слева направо, то чтение затрудняется, когда левая сторона текста плохо определена.

Выравнивание по центру

Каждая строка текста выравнивается по центру, и оба края оказываются неровными.

Выравнивание по ширине

Строки текста выровнены по левой и правой стороне окна браузера одновременно. Чем уже область вывода текста, тем больше вероятность появления пустого пространства внутри строк. Эта же особенность проявляется и при использовании длинных слов.
